# Functional Transcriptomics
Python lesson plans for working with transcriptomics data. Developed for [W22 BIOL SCI 378 - Functional Genomics at Northwestern University](https://biosci.northwestern.edu/courses/2019-2020/course-descriptions/) by Reese Richardson.

## Lessons

* [Lesson 1: Intro to Pandas](https://github.com/reeserich/functional_transcriptomics/blob/main/1_intro_to_pandas/intro_to_pandas_biol_sci_378_2022_01_21.ipynb)
* [Lesson 2: Normalization and clustering](https://github.com/reeserich/functional_transcriptomics/blob/main/2_normalization_and_clustering/normalization_and_clustering_220128_class_copy.ipynb)
* [Lesson 3: Over-representation analysis (ORA)](https://github.com/reeserich/functional_transcriptomics/blob/main/3_over_representation_analysis/over_enrichment_analysis_220211_class_copy.ipynb)
* [Lesson 4: Gene set enrichment analysis (GSEA)](https://github.com/reeserich/functional_transcriptomics/blob/main/4_gsea/gsea_220218.ipynb)
* [Bonus Lesson: k-Means clustering](https://github.com/reeserich/functional_transcriptomics/blob/main/9_k_means_demo/k_means_demo_220202.ipynb)
